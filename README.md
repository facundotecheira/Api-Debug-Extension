# 🐛 Debug Consola

Extensión de Chrome para debugging de peticiones HTTP.

## ✨ Características

- 📋 **Tabla de peticiones**: Visualizá todas las peticiones POST, PUT y DELETE en tiempo real
- 🔍 **Filtro por URL**: Buscá peticiones específicas
- 📦 **Agrupación automática**: Agrupa peticiones repetidas
- ✏️ **Editor integrado**: Modificá URL, headers y body y reenviá peticiones
- 🔄 **Comparador JSON**: Compará dos JSON fácilmente
- 🎯 **Análisis de campos null**: Detectá campos null y comparalos con un schema de referencia
- 📋 **Copiar al portapapeles**: Copiá respuestas con un clic

## 🚀 Instalación

1. Descargá el código o cloná el repositorio
2. Andá a `chrome://extensions/`
3. Activá "Modo desarrollador"
4. Hacé clic en "Cargar descomprimida"
5. Seleccioná la carpeta del proyecto

## 📝 Uso

1. Abrí DevTools (F12)
2. Buscá la pestaña "Debug Consola"
3. Hacé peticiones desde tu aplicación y aparecerán en la tabla
4. Hacé clic en cualquier petición para editarla
5. Modificá lo que necesites y reenviá
